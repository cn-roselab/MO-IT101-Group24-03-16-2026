# MotorPH Payroll System

## Project Description
The MotorPH Payroll System is a Java-based program that processes employee information and attendance records to compute payroll. The system reads data from CSV files and calculates employee salary based on attendance and employee details.

## Files Included
- MotorPHemployees.csv – contains employee information such as employee ID, name, and salary details.
- MotorPHemployeeattendance.csv – contains employee attendance records used to calculate working hours.
- MotorPHPayroll.iml – IntelliJ project file.

## Tools Used
- Java
